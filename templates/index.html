Perform a request for one of those routes to get application functionalities:

  • http://127.0.0.1:7700/stock - Return all data in Dict
  • http://127.0.0.1:7700/stock/section - Return a specific session from dict
  • http://127.0.0.1:7700/stock/section/product - Return the quantity in stock from a specific product in dict
  • http://127.0.0.1:7700/stock/section - POST Request - Insert some data in the dict
  • http://127.0.0.1:7700/stock/section - PUT Request - Create or replace data in dict
  • http://127.0.0.1:7700/stock/section - PATCH Request - Create or update data in dict
  • http://127.0.0.1:7700/stock/section - DELETE Request - Delete section in dict
  • http://127.0.0.1:7700/stock/section/product - DELETE Request - Delete product in dict
